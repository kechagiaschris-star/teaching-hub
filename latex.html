Βήμα-βήμα χρήση
  1. Επίλεξε τύπο εγγράφου και μάθημα
  2. Επικόλλησε κείμενο από Word/PDF ή περίγραψε τι θέλεις
  3. Πρόσθεσε επιπλέον οδηγίες αν χρειάζεται
  4. Πάτα Μετατροπή σε LaTeX
  5. Κατέβασε το .tex αρχείο
  6. Άνοιξε σε Overleaf ή τοπικό LaTeX
Γρήγορη σύνταξη μαθηματικών
$x^2 + 1$ → inline τύπος
\[ ... \] → centered τύπος
\frac{a}{b} → κλάσμα
\sqrt{x} → ρίζα
\sin, \cos, \tan → τριγ. συν.
\alpha, \beta, \theta → ελληνικά
\leq, \geq, \neq → σύμβολα
\in, \mathbb{R} → σύνολα
^{\circ} → μοίρες (°)
Overleaf — Setup (μια φορά)
1. Πήγαινε στο overleaf.com → Δωρεάν λογαριασμός
2. New Project → Blank Project
3. Αντικατέστησε το main.tex με το template
4. Menu (πάνω αριστερά) → Compiler → XeLaTeX (για ελληνικά)
5. Πάτα Recompile → PDF έτοιμο

💡 Tip: Χρησιμοποίησε XeLaTeX για καλύτερη υποστήριξη ελληνικών. Αν χρησιμοποιείς pdflatex, σιγουρέψου ότι έχεις το babel με greek.
Τοπική εγκατάσταση
Windows: MiKTeX + TeXstudio (δωρεάν)
Mac: MacTeX + TeXShop
Linux: sudo apt install texlive-full texstudio

Compile: xelatex myfile.tex ή pdflatex myfile.tex
